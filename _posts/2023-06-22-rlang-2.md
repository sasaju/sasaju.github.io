# 向量（Vector）
**向量**是用于存储数值型、字符型或逻辑型数据的一维数组。使用函数`c()`即可创建向量.\
## 创建向量
```R
a <- c(1, 2, 5, 7) 
b <- c("a", "c", 2) 
```
值得一提的是不同其他大多数编程语言，向量的索引（Index）不是从0开始而是从1开始

### 向量中包含不同类型的值
数值型即想1、2、3、1.2、3.6这种数字；字符型（"a", "b","asdf"）；逻辑型（TURE/FALSE）。
下面代码虽然可以执行，但不是最佳实践

```R
# 不建议使用
a <- c(1,"2","3")  
print(a)            #输出：[1] "1" "2" "3"

```
